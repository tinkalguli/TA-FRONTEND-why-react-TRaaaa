#### Basic Info

CDN url for babel to use in our project.

```html
<script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
```

While suing Babel don't forget to change `type` attribute of `script` element.

```html
<script type="text/babel">
  // JSX code goes here
</script>
```
